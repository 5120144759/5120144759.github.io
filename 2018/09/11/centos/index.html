Liunx常用命令

文件与目录操作

命令 解析
cd /home 进入 ‘/home’ 目录
cd.. 返回上级目录
cd../.. 返回上两级目录
cd - 返回上次所在目录
cp file1 file2 将file1复制为file2
cp -a dir1 dir2 复制一个目录
cp -a /tmp/dir1 复制一个目录到当前工作目录（.代表当前目录）
ls 查看当前目录下所有文件
ls -a 显示隐藏文件
ls -l 显示详细信息
ls -lrt 按时间显示文件（l表示详细列表，r表示反向排序，t表示按时间排序）
pwd 显示当前路径
mkdir dir1 创建 ‘dir1’ 目录
mkdir dir1 dir2 创建2个目录
mkdir -p /tmp/dir1/dir2 创建一个目录树
mv dir1 dir2 移动/重命名一个目录
rm -f file1 删除 ‘file1’
rm -rf dir1 删除 ‘dir1’ 目录及其子目录内容
ln -s /etc/centos-release /root/sysinfo 创建一个软链接

查看文件内容

命令 解析
wc 统计一个文件里行数，单词数，字节数
paste file1 file2 将文件1和文件2合并
cut -b 4-9 file 将文件中的第4-9个字节截取出来
chmod 改变文件权限
chown 修改文件控制权
yum -y xxx 安装的时候直接回答yes

其他命令

命令 解析
ps 看到正在运行进程
ps -ef/ps -aux 看到所有正在运行的进程
kill 进程号 杀死进程(给善后时间)
kill -9 进程号 强行杀死进程
yum remove 删除安装程序
yum list installed 查看已安装程序
yum search 搜索
yum check-update 检查更新
yum install 安装软件
yum info 查看信息
yum update 更新
rpm -ivh .rpm 安装
rpm -e 卸载
rpm -qa 查看
xargs rpm -e 将前面的输出作为后面的输入

alias shit=’ps -ef | grep redis | awk ‘{print $2}’ | xargs kill’ 将命令别名

centos安装Python3

  • wget https://www.python.org/ftp/python/3.6.5/Python-3.6.5.tgz 下载python包

  • tar -zxvf Python-3.6.5.tgz 解压python包

  • yum install gcc 安装c语言编译器

  • yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel 安装Python可能需要的依赖库

  • ./configure –prefix=/usr/local/python36 –enable-optimizations 生成Makefile文件

  • make && make install 构建和安装

  • ln -s /usr/local/python36/bin/python3.6 /usr/bin/python3

  • ln -s /usr/local/python36/bin/pip3 /usr/bin/pip3

  • ln -s /usr/local/python36/bin/2to3-3.6 /usr/bin/2to3
    做软连接

  • python3 -m pip install -U pip 更新pip包管理工具
