mysql
安装mysql
  • 安装mysql服务
    sudo apt-get install mysql-server
  • 安装客户端
    sudo apt install mysql-client
  • 安装依赖
    sudo apt install libmysqlclient-dev
修改用户权限
  • 删除旧用户DROP USER ‘root‘@’localhost’;
  • 添加新用户CREATE USER ‘root‘@’%’ IDENTIFIED BY ‘123456’;
  • 添加权限GRANT ALL PRIVILEGES ON . TO ‘root‘@’%’ WITH GRANT OPTION;
  • FLUSH PRIVILEGES;
修改字符集

sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf

在[mysqld]段落中添加如下两行：

character-set-server=utf8

collation-server=utf8_general_ci

安装和配置postgresql

1、安装postgresql

sudo apt-get install postgresql

安装完毕后，系统会创建一个数据库超级用户“postgres”, 密码为空。这个用户既是不可登录的操作系统用户，也是数据库用户

2、切换到Linux的postgres用户下

sudo su postgres

3、登陆postgresql

psql postgres

4、设置密码

ALTER USER postgres with PASSWORD ‘postgres’

5、允许任何地方链接

vim /etc/postgresql/9.1/main/postgresql.conf

#listen_addresses = ‘localhost’改为 listen_addresses = ‘*’

6、启用密码验证

#password_encryption = on 改为 password_encryption = on

7、可访问的用户ip段

vim /etc/postgresql/9.1/main/pg_hba.conf末尾添加

to allow your client visiting postgresql server

host all all 0.0.0.0 0.0.0.0 md5

安装pgAdmin3

sudo apt-get install pgadmin3

pgadmin3运行

rpm与deb转化工具alien

sudo apt-get install alien

sudo alien –scripts -d oracle-xe-11.2.0-1.0.x86_64.rpm
